## 一、整体视觉风格规范（已锁定）

### 风格定义
**RMBTI Oracle Low-Poly Style（老钱神谕牌风格）**

| 维度 | 规范 |
|---|---|
| 人物风格 | 16personalities MBTI 成人低多边形（Low-Poly）风格，头身比约 1:3，几何切面感，非 Q 版 |
| 人物性别 | 明显男性：方下颌、宽肩、短发、几何面 |
| 服装标准 | 老钱风深色西装：金边翻领、白口袋巾、金袖扣、金领针/领带夹，无暴发户金链 |
| 表情标准 | 得意迷人、嘴角微扬，"我知道你喜欢"的自信笑，**禁止苦情/卑微/渴求表情** |
| 姿态标准 | 老板永远是**给予者/高位者**，姿态是赏赐/掌控/压场，**禁止仰头求人/双手捧物的卑微姿势** |
| 道具标准 | 所有道具**夸张化处理**，比正常尺寸大 1.5-2 倍，带发光/金光效果 |
| 卡框标准 | Art Deco 金线牌框，底部人格名称金色大写字母 |
| 构图标准 | 人物居中偏上，道具/场景元素填满画面，底部留文字区 |
| 比例 | 2:3 竖版卡牌比例 |

## 二、已确认定稿卡面

### 1. 君临 KING ✅ 定稿
- **文件**：`rmbti_king_final.png`
- **主色调**：深黑 #0A0A0F + 琥珀金
- **标志性元素**：悬浮超大几何金冠（光线构成，非实体）、穹顶拱廊大厅、黑色大理石台、下方一排仰视剪影人群
- **戏剧动作**：双手背后，身体直立，下颌微抬，一手轻垂——不需要动作，存在即是压场
- **核心感受**：他不需要宣布自己，全场已经知道

### 2. 深情 DEEP ✅ 定稿
- **文件**：`rmbti_deep_v6.png`
- **主色调**：深海蓝 #0D1B2A + 暖香槟金 + 玫瑰红
- **标志性元素**：超大发光红玫瑰（比手掌大）、单束聚光灯从上方打下、四周人群全在暗处、飘浮大号心形光点
- **戏剧动作**：单手平托玫瑰向前递出（赏赐姿态），另一手自然垂侧，身体微微后倾
- **服装细节**：深蓝西装金边、白口袋巾、金领针、白袖口金袖扣
- **核心感受**：他不需要开口，玫瑰替他说话；聚光灯只找他一个人

### 3. 逆转 SAVE-R ✅ 定稿
- **文件**：`rmbti_saver_v1.png`
- **主色调**：深红 #3D0000 + 金色
- **标志性元素**：超大金色上升箭头（背景）、棋子/筹码/金币炸飞满画面、棋盘被推翻
- **戏剧动作**：身体大步前冲，一臂横扫推翻棋局，棋子飞散，另一手指向前方
- **核心感受**：他一出现，结局就改写了；他不是来陪跑的，是来改命的

### 4. 掌盘 CTRL ✅ 待确认
- **文件**：`rmbti_ctrl_v1.png`
- **主色调**：深绿 #0D2B1A + 金色
- **标志性元素**：超大发光金色方向盘（几何线构成）、金色路线光轨向四面放射、金钥匙飞散
- **戏剧动作**：双手握住方向盘，双脚分开站稳，身体微前倾，眼神锐利聚焦
- **核心感受**：今晚往哪走，由他说了算

## 四、设计标准三原则（核心）

1. **老板永远是高位者** — 给予、掌控、压场，从不渴求、仰视、等待
2. **表情永远是得意迷人** — 自信微笑，"我知道你喜欢"，从不苦情
3. **道具永远是夸张的** — 比正常大 1.5-2 倍，带光效，一眼能读懂行为
