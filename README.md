# RMBTI 人民币人格测试

面向语音房/直播场景的"老板人格测试"H5 页面。

## 产品定位

不是心理学量表，而是一个**社交消费人格剧场**。测试用户花钱时最想买到什么感觉，以及通常怎么出手。

## 结果结构

- **主牌人格**（8 种）：深情 / 逆转 / 掌盘 / 长情 / 神话 / 典藏 / 君临 / 绝杀
- **副签出手方式**（4 种）：爆冲 / 常陪 / 豪侠 / 卡点
- **镜面标签**：润色结果页的风格标签

最终展示：**主牌·副签**（如"君临·卡点"）

## 技术栈

HTML + CSS + JavaScript，配置驱动，便于后续接入业务系统。

## 本地运行

直接用浏览器打开 `src/index.html`。

计分引擎可用 Node 做轻量验收：

```bash
node src/engine.test.js
```
